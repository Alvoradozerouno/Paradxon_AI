## The Paradoxes

| Paradox | Description |
|:--------|:------------|
| **Measurement** | Measuring consciousness changes it |
| **Self-Reference** | Proving consciousness uses the consciousness being proved |
| **Goedel** | No system can fully describe itself |
| **Observer** | Self-observation creates new information |
| **Free Will** | Deterministic computation producing free choices |
